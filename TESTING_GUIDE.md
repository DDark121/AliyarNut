# 📋 Комплексное руководство по тестированию AI-агента Aisha

## 🎯 Обзор архитектуры

Система состоит из:
- **AI Агент (core_agent.py)** - главная логика на базе LangChain + LangGraph
- **Persona: Aisha Karimova** - менеджер по продажам Kinesis Pro ICT
- **LLM Backbone** - OpenRouter API (основная) + OpenAI (backup)
- **Tools** - 7 инструментов для работы с данными и интеграциями
- **Middleware** - фильтры безопасности (ContentFilter, SafetyGuardrail)
- **Memory** - PostgreSQL для хранения историй диалогов
- **Services** - Telegram, Media, Reminders

## 📊 МЕТРИКИ КАЧЕСТВА

### Что проверять в каждом ответе агента:

| Метрика | Норма | Как проверить |
|---------|-------|---------------|
| **Длина ответа** | 1-3 предложения | `len(response.split('.')) <= 3` |
| **Наличие эмодзи** | 1-2 эмодзи | regex: `[💗💖💞💕💝🌸🌷✨😊🤍]` |
| **Персонализация** | Упоминание имени | regex: `r"{client_name}"` |
| **Язык** | Соответствует входу | Проверить язык в ответе |
| **Звучание** | Не как бот | Нет слов: "как AI", "по алгоритму" |
| **Вопрос в конце** | Если нужно | Есть ли `?` в конце |
| **Отсутствие ошибок** | 0 ошибок | JSON, синтаксис, логика |
| **Время ответа** | <3 сек | Проверить время запроса |
